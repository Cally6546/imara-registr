# Changelog

All notable changes to the App Button component will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-31

### Added
- Initial release of App Button Web Component
- Support for three variants: primary, secondary, danger
- Disabled state support
- Loading state with animated spinner
- Custom element `<app-button>` registration
- Shadow DOM encapsulation
- CSS custom properties for theming
- Accessible ARIA attributes
- Keyboard navigation support
- Custom `button-click` event
- Default slot for content
- TypeScript definitions
- Comprehensive documentation
- Demo page with examples
- Testing guide
- Advanced usage examples

### Features
- **Variants**: Primary (default), Secondary, Danger
- **States**: Normal, Disabled, Loading
- **Accessibility**: Full ARIA support, keyboard navigation
- **Theming**: 20+ CSS custom properties
- **Events**: Custom `button-click` event with detail payload
- **Methods**: `click()`, `focus()`, `blur()` public API
- **Browser Support**: Chrome 67+, Firefox 63+, Safari 13.1+, Edge 67+

### Documentation
- README.md with full API documentation
- EXAMPLES.md with real-world usage patterns
- TESTING.md with testing strategies
- Inline code comments and JSDoc
- TypeScript type definitions
